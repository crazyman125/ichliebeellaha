# Ellahas Herzensquest

Eine personalisierte mobile Geburtstags-Website für Ellaha von Kamal.

## Inhalt
- `index.html` – Spielstruktur
- `style.css` – romantisches Design mit Herzfarben
- `script.js` – Spiellogik, Touch-Steuerung, Dialoge, Belohnungsanimationen

## Personalisierung
- Hauptfigur: **Ellaha**
- Sprecher in den Dialogen: **Kamal**
- Fragen:
  - Geburtstag: **10.03.1993**
  - Erstes Date: **14.06.2025**
  - Heirat: **24.08.2025**

## Features
- Smartphone-tauglich
- Große Touch-Steuerung
- Progressive Belohnungsanimationen bei richtigen Antworten
- Romantischer Look mit Herzen und Abschlussbotschaft
- Läuft direkt im Browser ohne externe Abhängigkeiten

## Start
Einfach die `index.html` öffnen.
