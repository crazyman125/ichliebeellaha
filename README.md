# Herzensquest – Geburtstagsabenteuer

Ein kleines romantisches Browser-Spiel im Retro-Stil für Smartphones und Desktop.

## Dateien
- `index.html` – Struktur der Seite
- `style.css` – romantisches Design, mobile Layouts, Animationen
- `script.js` – Spiellogik, Touch-Steuerung, Fragen, Belohnungen

## Features
- spielbar auf dem Smartphone
- große Touch-Buttons
- 3 NPCs mit 3 persönlichen Fragen
- romantischer Look mit Herzen
- Belohnungsanimationen, die sich bei jeder richtigen Antwort steigern
- Abschlussnachricht zum Geburtstag
- keine externen Bilder oder Bibliotheken nötig

## Start
`index.html` einfach im Browser öffnen oder auf einen Webserver legen.

## Personalisierung
In `script.js` kannst du Namen, Texte, Fragen, Antworten und die Abschlussbotschaft leicht anpassen.
