# Retro Quest – Mini RPG

Ein kleines mobiles Browser-Spiel im Retro-Stil, inspiriert von klassischen Top-Down-Abenteuern.

## Enthaltene Dateien
- `index.html` – Grundstruktur
- `style.css` – Retro-UI + mobile Responsiveness
- `script.js` – Spiellogik, Touch-Steuerung, Fragen-System

## Features
- Mobile-first Layout
- Touch-Steuerung mit großen Buttons
- Tastatursteuerung zusätzlich möglich
- 3 NPCs mit 3 Fragen
- Ergebnisanzeige am Ende
- Canvas-basiertes Spielfeld ohne externe Assets
- Fokus auf Performance, Lesbarkeit und einfache Deployment-Fähigkeit

## Starten
Einfach `index.html` im Browser öffnen oder auf einem Webserver hosten.
